      সিসকো এবং জুনিপার কমান্ড তুলনা

      1. কনফিগারেশন মোড:

      সিসকো IOS:

      conf t

      conf t কমান্ডটি কনফিগারেশন মোডে প্রবেশ করে।

      জুনোস OS:

      configure

      configure কমান্ডটি কনফিগারেশন মোডে প্রবেশ করে।

      2. BGP কনফিগারেশন:

      সিসকো IOS:

      router bgp 10
 bgp router-id 10.10.0.1
 neighbor IBGP-PEER peer-group
 neighbor IBGP-PEER remote-as 10
 neighbor IBGP-PEER update-source loopback0
 neighbor 10.10.0.2 peer-group IBGP-PEER
 address-family ipv4 unicast
  neighbor IBGP-PEER activate
  neighbor IBGP-PEER next-hop-self
  neighbor IBGP-PEER route-reflector-client
      • router bgp দিয়ে BGP প্রসেস শুরু হয়।
      • neighbor দিয়ে পিয়ার গ্রুপ এবং নেইবার কনফিগার করা হয়।
      • address-family দিয়ে IPv4 ইউনিকাস্ট অ্যাক্টিভেট করা হয়।

      জুনোস OS:

      set routing-options autonomous-system 10
set routing-options router-id 10.10.0.1
set protocols bgp group IBGP-PEER type internal
set protocols bgp group IBGP-PEER local-address 10.10.0.1
set protocols bgp group IBGP-PEER neighbor 10.10.0.2
set protocols bgp group IBGP-PEER family inet unicast
set protocols bgp group IBGP-PEER cluster 10.10.0.1
      • set routing-options autonomous-system দিয়ে AS নাম্বার সেট করা হয়।
      • set protocols bgp group দিয়ে BGP গ্রুপ তৈরি করা হয়।
      • family inet unicast দিয়ে IPv4 ইউনিকাস্ট অ্যাক্টিভেট করা হয়।
      • cluster দিয়ে Route Reflector ক্লাস্টার আইডি সেট করা হয়।

      3. নেক্সট-হপ সেলফ:

      সিসকো IOS:

      neighbor IBGP-PEER next-hop-self

      next-hop-self কমান্ডটি ব্যবহার করে নেক্সট হপ হিসেবে নিজেকে advertise করা হয়।

      জুনোস OS:

      set protocols bgp group IBGP-PEER export NEXT-HOP-SELF
set policy-options policy-statement NEXT-HOP-SELF term 1 then next-hop self

      next-hop self পলিসি তৈরি করে নেক্সট হপ হিসেবে নিজেকে advertise করা হয়।

      4. রুট রিফ্লেক্টর ক্লায়েন্ট:

      সিসকো IOS:

      neighbor IBGP-PEER route-reflector-client

      route-reflector-client কমান্ডটি ব্যবহার করে ক্লায়েন্ট রাউটারকে Route Reflector ক্লায়েন্ট হিসেবে কনফিগার করা হয়।

      জুনোস OS:

      set protocols bgp group IBGP-PEER cluster 10.10.0.1

      cluster কমান্ডটি ব্যবহার করে Route Reflector ক্লাস্টার আইডি সেট করা হয়।

      সিসকো এবং জুনিপারে LACP কনফিগারেশন পার্থক্য

      সিসকো কনফিগারেশন:

      ইথারচ্যানেল (Port-channel) কনফিগারেশন:

      interface Port-channel3
  description ACCESS-RW-LINK-LACP
  switchport mode trunk
  switchport trunk allowed vlan 1-4049

      ফিজিক্যাল ইন্টারফেসে LACP কনফিগারেশন:

      interface Ethernet1/5
  description ACCESS-RW-LINK
  switchport mode trunk
  switchport trunk allowed vlan 1-4049
  channel-group 3 mode active

      জুনিপার কনফিগারেশন:

      অ্যাগ্রিগেটেড ইথারনেট (ae) ইন্টারফেস কনফিগারেশন:

      set interfaces ae3 description "ACCESS-RW-LINK-LACP"
set interfaces ae3 unit 0 family ethernet-switching interface-mode trunk
set interfaces ae3 unit 0 family ethernet-switching vlan members 1-4049

      ফিজিক্যাল ইন্টারফেসে LACP কনফিগারেশন:

      set interfaces ge-0/0/5 description "ACCESS-RW-LINK"
set interfaces ge-0/0/5 ether-options 802.3ad ae3
